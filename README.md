# Poumau Official Page (Static)

This is the official landing page for **Poumau**, a premium minimal brand.

## Features
- Minimalist design using Pretendard font.
- Responsive layout (Desktop, Tablet, Mobile).
- Interactive Hero slider using Swiper.js.
- Image overview modal for product details.
- SEO optimized.

## Structure
- `index.html`: Main landing page containing all HTML, CSS, and JS.
- `assets/img/`: Directory for images (to be populated).

## Development
Since this is a static site, you can simply open `index.html` in any browser to see the results.

## Deployment
This project is configured to be deployed on static hosting services (e.g., Cloudflare Pages, GitHub Pages, Vercel).
